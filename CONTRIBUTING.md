# Contributing

Thanks for improving Chapter Reader.

1. Fork and clone the repo.
2. Run `./setup.sh` and verify with `.venv/bin/python discover.py`.
3. Make focused changes; keep the app local-only (`127.0.0.1`).
4. Open a PR with what you tested (Mac version, engine used).

**Scope:** offline TTS, readability, setup reliability — not cloud features or account systems.
